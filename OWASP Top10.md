OWASP(Open Web Application Security Project ) is global non-profit foundation that provides resources and tool to ensure the web application security. OWASP top 10 is they list the top 10 overall threats/vulnerabilities happened in the last 3-4 years. 

As per recent updated the OWASP Top 10(2025) is. 

### 1. A01:2025 - Broken Access Control :-

It happpens when a web application fails to properly restricts what an user can access and do. 

Example : Login as a normal user and change to admin and get full access. 

### 2. A02:2025 - Security Misconguration :-

It happens when a web application, server or system set up in a way that leaves it exposed to attackers. 

Example : Reveals too much in error messages, insecure http request, outdated software, default credentials unchanges like password : password
